COMPENDIUM

WHAT IS IT

Compendium is a static library which include a configurator for edit files.
The library include:
  • Two class for control data
  • Class configurator for control a file
The library availalbe for Windows only, no plans to port it. Compendium files can be structured or flat (only units). Take a look at examples below: 
<- Flat structure ->
unit FirstName:Serge
unit SecondName:Page
 flat structure 
group Character [
  <-
  This is a configuration
  of the fictional character
  ->

  unit FirstName:Serge
  unit SecondName:Page
  unit Age:21

  group Equipment [
    <-
    Equipment:
    0 - Empty
    1 - Pistol
    2 - Shotgun
    3 - Rifle
    ->

    unit LeftHand:2
    unit RightHand:0
  ]
]
 classic structure

PROJECT GOAL

The main goal - create a structured configuration format which can be pretty useful for store settings of characters in a game or another purposes but only for store.
